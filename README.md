Vijay Oliver – Portfolio Website

🌐 Live Website: https://vijayoliver.github.io/vijay-portfolio/

This is my personal developer portfolio website built completely from scratch using HTML, Tailwind CSS, and JavaScript.
The purpose of this website is to showcase my skills, projects, creative UI design, and development capabilities.

The site focuses on modern UI/UX design, responsiveness, and performance, providing visitors with an engaging and visually attractive experience.

✨ Features:

🌗 Light Mode & Dark Mode Toggle
⚡ Fast and Lightweight Static Website
📱 Fully Responsive Design
🎨 Modern Attractive UI/UX
📩 Working Enquiry Form using EmailJS
💬 WhatsApp Direct Contact Integration
🚀 Deployed with GitHub Pages
🎯 Clean and Organized Code Structure
💻 Pure Frontend (No Backend Required)
🛠️ Technologies Used

Frontend:
HTML5
Tailwind CSS
JavaScript

Integrations:
EmailJS – For sending enquiry form messages directly to email
WhatsApp API – For instant contact via WhatsApp

Hosting:
GitHub Pages
